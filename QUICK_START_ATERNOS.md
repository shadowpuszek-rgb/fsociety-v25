# FSOCIETY V25 - Quick Start Guide for Aternos

## 🚀 Fastest Way to Play on Aternos

### Step 1: Install Datapack (Required)
1. Download `fsociety-v25` folder from GitHub
2. Go to your Aternos server files
3. Upload to: `world/datapacks/fsociety-v25/`
4. Restart server
5. Run command: `/reload`

✅ **Game is fully playable at this point!**

### Step 2: Add Resource Pack (Optional - Better Experience)
1. Download `resourcepack.zip` from GitHub releases
2. Place in `.minecraft/resourcepacks/` **on your client machine**
3. Enable in Minecraft launcher
4. Join server - textures will show

## 🆘 Troubleshooting

### Datapack not loading
- Check folder structure: `world/datapacks/fsociety-v25/`
- Run `/datapack list` to verify
- Restart server with `/reload`

### Commands not working
- Confirm datapack is loaded
- Check your permission level (need level 2+)
- Verify spelling of command

### Resource pack not showing
- Install locally on your client (not server)
- Enable in resource pack menu
- Restart Minecraft completely
- Check Downloads folder

### Sounds not playing
- Verify resource pack is enabled
- Check Minecraft volume settings
- Update audio drivers

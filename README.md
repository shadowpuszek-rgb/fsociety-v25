## 📦 Installation

### Requirements
- Minecraft Java Edition 1.21.11+
- Optional: Aternos plugins (ProtocolLib, Citizens, WorldEdit, Vault)

### Steps
1. Download `fsociety-v25.zip` from releases
2. Extract to `world/datapacks/`
3. Run `/reload` in-game
4. Craft your first laptop to begin

## 🚀 Getting Started

### First Mission
1. Craft a Hacking Laptop (4 copper blocks, 1 amethyst shard, 1 redstone block, 2 iron ingots)
2. Hold the laptop and right-click to boot
3. Select your operating system
4. Accept the first mission from fsociety
5. Start with basic hacking techniques

### Early Game Progression
- **Level 1-5:** Learn basic hacking (Brute Force, SQL Injection)
- **Level 5-15:** Craft advanced gear (Flipper Zero, Bad USB, Drone)
- **Level 15-30:** Join a faction and gain reputation
- **Level 30+:** Unlock endgame content (Metasploit, Hashcat)

## 🛠️ Commands

```mcfunction
# Give yourself a laptop
/give @s written_book{fsociety_laptop:1b}

# Join DedSec
/function fsociety:factions/dedsec/join

# Join fsociety
/function fsociety:factions/fsociety/join

# Craft Bad USB
/function fsociety:gear/bad_usb/craft

# Deploy drone
/function fsociety:gear/drone/deploy

# Enable TOR
/function fsociety:security/tor/enable

# Launch Metasploit
/function fsociety:hacking/metasploit/launch

# Start police raid event
/function fsociety:events/police_raid
```

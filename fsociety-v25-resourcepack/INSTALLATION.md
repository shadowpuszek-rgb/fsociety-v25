# FSOCIETY V25 Resource Pack - Installation Guide

## Installation Steps

1. **Download the Resource Pack**
   - Download `fsociety-v25-resourcepack.zip` from releases
   - Do NOT extract it

2. **Locate Resource Packs Folder**
   - Windows: `.minecraft/resourcepacks/`
   - Mac: `~/Library/Application Support/minecraft/resourcepacks/`
   - Linux: `~/.minecraft/resourcepacks/`

3. **Place the ZIP File**
   - Copy `fsociety-v25-resourcepack.zip` into the resourcepacks folder
   - Keep it as a ZIP file (don't extract)

4. **Enable in Minecraft**
   - Open Minecraft
   - Go to Options → Resource Packs
   - Find "FSOCIETY V25 Resource Pack" in the available list
   - Click arrow to move it to the active packs
   - Click "Done"

5. **Restart Game**
   - Restart Minecraft for changes to take effect
   - Create new world or join server with datapack

## Troubleshooting

### Resource pack not showing
- Ensure file is `.zip` format (not extracted)
- Check folder path is correct
- Restart Minecraft
- Update to latest Minecraft version

### Textures look wrong
- Delete resource pack cache: `.minecraft/resourcepacks/.index/`
- Restart game
- Re-enable resource pack

### Sounds not playing
- Check volume settings in Minecraft
- Verify sound files are present
- Update graphics drivers

### Font issues
- Ensure `assets/fsociety/fonts/` files exist
- Check font PNG dimensions (256x256)
- Restart game after installing

## Customization

You can customize the resource pack:

1. **Extract the ZIP** to a folder
2. **Edit textures** in `assets/minecraft/textures/`
3. **Modify fonts** in `assets/fsociety/fonts/`
4. **Change colors** in texture files
5. **Re-zip the folder** keeping the structure
6. **Place back in resourcepacks folder**
